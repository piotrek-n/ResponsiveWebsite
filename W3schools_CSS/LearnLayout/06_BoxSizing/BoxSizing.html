Poprzednie rozwiązanie nie było nieintuicyjne, dlatego można użyć box-sizing

The original box model behavior was eventually considered nieintuicyjne, so a new CSS property called box-sizing was created. When you set box-sizing: border-box; on an element, the padding and border of that element no longer increase its width.
